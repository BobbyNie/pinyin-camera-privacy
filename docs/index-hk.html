常見問題

1. 拼音相機是否需要連網？
不需要。主要功能可完全離線使用。
2. 圖片是否會上傳？
不會。圖片辨識和拼音標註均在裝置本地完成，不會上傳到任何伺服器。
3. 是否有廣告或訂閱？
沒有廣告，無訂閱，為一次性購買。
4. 如何儲存辨識後的圖片？
辨識完成後，點擊儲存按鈕即可將圖片儲存到您的相簿。
5. 支援從相簿匯入圖片嗎？
支援。您可以從相簿中選擇圖片進行辨識。
