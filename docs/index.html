常见问题

1. 拼音相机是否需要联网？
不需要。主要功能可完全离线使用。
2. 图片是否会上传？
不会。图片识别和拼音标注均在设备本地完成，不会上传到任何服务器。
3. 是否有广告或订阅？
没有广告，无订阅，为一次性购买。
4. 如何保存识别后的图片？
识别完成后，点击保存按钮即可将图片保存到您的相册。
5. 支持从相册导入图片吗？
支持。您可以从相册中选择图片进行识别。
